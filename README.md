# Credit Card Statement PDF Parser

📖**Summary**
Credit Card Statement PDF Parser is a Python-based tool that simplifies the extraction of transaction data from credit card PDF statements. Using a Streamlit interface, users can easily upload their statements, parse the transactions, and export the results as a structured CSV file. The project combines PDF reading utilities and parsing logic to automate data extraction, making it easier to analyze spending patterns or integrate with personal finance tools. It’s lightweight, interactive, and designed for non-technical users while remaining extendable for developers.

🧩**Installation and Setup:**

1️⃣**Clone the Repository**

git clone https://github.com/vedantidabholkar/credit-card-statement-parser.git
cd credit-card-statement-parser


2️⃣**Install Dependencies**

pip install -r requirements.txt


3️⃣**Run the App**

streamlit run app.py
